h2. Intro to Textile

All posts on this website uses the "Textile":http://en.wikipedia.org/wiki/Textile_(markup_language) markup language, it turns texts into html, so it's very easy to use.  Here is a Textile "sandbox.":http://www.textism.com/tools/textile/ for you to test the look & feel of your input.

<hr />

h2(icon-camera-retro). &#160;Adding images in textile

In order for a image to show up in your posting, you need to 

# upload it
# link the uploaded file to your article (which is being written in Textile) 

h3. Uploading the Image Files 

This had been covered in "setting up your environment":set_up_environment.html

h3. Linking Image Files to Post

This is how to add images

@!/path/to/my/image.jpg(my cool image description)!@

This is how to add an image that floats to the right hand side

@!(img-right)/path/to/my/image.jpg(my cool image description)!@

h2(icon-film). &#160;Adding video in textile

In order for a video to show up in your posting, you need to 

# upload it to Vimeo (we use Vimeo to host the videos)
# link the uploaded file to your article (which is being written in Textile) 

h3. Uploading the Video Files 

# upload the video file to vimeo
# set the sharing setting, so that everyone on the internet can view the video
# copy the video number for the video so that you can link it

h3. Linking Video Files to Post

Once the video is uploaded, we need to link the video the the post that you're writing in Textile.  This is how you'd do it.  Please be sure to change the video number to the one that you've just uploaded. In the example below, the video number is __39943514__.  If you prefer the video to show up bigger on your posting, just change the width & height attributes below.

@<iframe src="http://player.vimeo.com/video/39943514?portrait=0" width="400" height="300" frameborder="0" ></iframe>@

h3. Linking Document Files to Post

Please be sure to change the document id to the one that you've just uploaded. In the example below, the doc's id is __0B5-xkEf58AbIUGVVanhnd3E4M1U__.
